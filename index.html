RESERCH PROJECT

Identification of diabetic retinopathy using retinal images:

Compled a project on the topic Identification Of Diabetic Retinopathy Using Retinal Images under the guidance of Mrs. Darling Jemima.
A Deep Learning algorithm using Convolutional Neural Network in Python language was developed to predict the severity of Diabetes(0-4) of a person using the digital retinal images.
Using the dataset present in the Kaggle website, a training algorithm was developed to train the model. The models that were used during this process are ShuffleNet, EfficientNet, VGG16 and AlexNet and the efficiency between these models are analysed. The predicted algorithm was also developed for finding severity of Diabetes of a person in real time using digital retinal image with the help of the trained models.
